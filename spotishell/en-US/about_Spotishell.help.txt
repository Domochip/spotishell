TOPIC
    about_Spotishell

SHORT DESCRIPTION
    A powershell module used to easily interact with the Spotify API

EXAMPLES
    Search-Spotify -Query "Bloc Party" -Artist
    New-SpotifyApplication -Name "blah" -ClientId "myclientId" -ClientSecret "myClientSecret"
